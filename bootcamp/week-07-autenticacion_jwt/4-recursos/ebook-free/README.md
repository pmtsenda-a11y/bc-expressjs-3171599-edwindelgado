# 📚 Ebooks Gratuitos — Semana 07: Autenticación JWT

## Recursos Disponibles

### 1. Node.js Security Handbook (Liran Tal — Snyk)
**Enlace**: https://www.snyk.io/wp-content/uploads/snyk-nodejs-security-handbook.pdf
**Relevancia**: Cubre bcrypt, JWT, cookies seguras y OWASP Top 10 en aplicaciones Node.js.

### 2. OWASP Testing Guide v4.2 (secciones de autenticación)
**Enlace**: https://owasp.org/www-project-web-security-testing-guide/
**Relevancia**: Capítulos sobre testing de autenticación, gestión de sesiones y enumeración de usuarios.

### 3. JWT Handbook (Sebastián Peyrott — Auth0)
**Enlace**: https://auth0.com/resources/ebooks/jwt-handbook
**Relevancia**: Guía completa de JWT: estructura, algoritmos, vulnerabilidades y mejores prácticas. Cubre HS256 vs RS256, `alg:none` attack, y token revocation.

### 4. Documentación oficial: jsonwebtoken (npm)
**Enlace**: https://www.npmjs.com/package/jsonwebtoken
**Relevancia**: API completa de `jwt.sign()`, `jwt.verify()`, manejo de errores y opciones de configuración.

### 5. Documentación oficial: bcrypt (npm)
**Enlace**: https://www.npmjs.com/package/bcrypt
**Relevancia**: API asíncrona vs síncrona, salt rounds, integración con Node.js y notas de seguridad.
