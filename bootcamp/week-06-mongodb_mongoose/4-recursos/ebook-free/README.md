# 📚 eBooks Gratuitos — Semana 06: MongoDB + Mongoose

Recursos de lectura gratuitos recomendados para profundizar en los temas de esta semana.

---

## MongoDB

### MongoDB Manual (Documentación Oficial)
- **URL**: https://www.mongodb.com/docs/manual/
- **Formato**: Web (HTML)
- **Descripción**: Documentación oficial completa de MongoDB. Incluye conceptos de modelado, indexación, aggregation pipeline, operadores de consulta y administración. Referencia definitiva para cualquier duda técnica.
- **Secciones clave**:
  - [Introduction to MongoDB](https://www.mongodb.com/docs/manual/introduction/)
  - [Documents](https://www.mongodb.com/docs/manual/core/document/)
  - [Data Modeling](https://www.mongodb.com/docs/manual/data-modeling/)
  - [Indexes](https://www.mongodb.com/docs/manual/indexes/)
  - [Aggregation](https://www.mongodb.com/docs/manual/aggregation/)

### MongoDB University — Cursos Gratuitos
- **URL**: https://learn.mongodb.com/
- **Formato**: Cursos interactivos online
- **Descripción**: Plataforma oficial de aprendizaje de MongoDB con cursos completamente gratuitos. Incluyen badges y certificaciones. Los más relevantes para esta semana:
  - **MongoDB and the Document Model** (fundamentos)
  - **Getting Started with MongoDB Atlas** (cloud)
  - **MongoDB CRUD Operations in Node.js**

### The Little MongoDB Book (v2)
- **URL**: https://github.com/nicholasjackson/the-little-mongodb-book
- **Formato**: PDF / Markdown
- **Descripción**: Guía introductoria concisa (~50 páginas) que cubre los conceptos fundamentales de MongoDB: documentos, colecciones, CRUD, indexación y mapa-reduce. Ideal para leer en 2-3 horas.

## Mongoose

### Mongoose Official Guide
- **URL**: https://mongoosejs.com/docs/guide.html
- **Formato**: Web (HTML)
- **Descripción**: Documentación oficial de Mongoose. Cubre en detalle Schemas, Models, Queries, Populate, Middleware, Validation y TypeScript integration. Recurso principal para resolver dudas de implementación.
- **Secciones clave**:
  - [Schemas](https://mongoosejs.com/docs/guide.html)
  - [Models](https://mongoosejs.com/docs/models.html)
  - [Queries](https://mongoosejs.com/docs/queries.html)
  - [Populate](https://mongoosejs.com/docs/populate.html)
  - [Validation](https://mongoosejs.com/docs/validation.html)
  - [Middleware](https://mongoosejs.com/docs/middleware.html)
  - [TypeScript Support](https://mongoosejs.com/docs/typescript.html)

### Mongoose TypeScript Guide
- **URL**: https://mongoosejs.com/docs/typescript.html
- **Formato**: Web (HTML)
- **Descripción**: Guía específica de Mongoose para TypeScript. Explica cómo definir interfaces, generic types en Schema, y cómo tipar correctamente `populate()` con resultados hidratados.

## Docker

### Docker Hub — Imagen oficial de MongoDB
- **URL**: https://hub.docker.com/_/mongo
- **Formato**: Web (HTML)
- **Descripción**: Página oficial de la imagen Docker de MongoDB. Documenta variables de entorno (`MONGO_INITDB_ROOT_USERNAME`, `MONGO_INITDB_DATABASE`), volúmenes, scripts de inicialización y configuración de autenticación.
