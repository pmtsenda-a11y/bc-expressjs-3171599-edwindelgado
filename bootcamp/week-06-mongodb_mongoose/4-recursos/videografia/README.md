## MongoDB Fundamentos

### MongoDB Crash Course 2022 — Traversy Media
- **URL**: https://www.youtube.com/watch?v=-56x56UppqQ

- **Idioma**: Inglés

### MongoDB in 100 Seconds — Fireship
- **URL**: https://www.youtube.com/watch?v=-bt_y4Loofg
- **Duración**: ~2 minutos
- **Idioma**: Inglés

## TypeScript + Mongoose

### Mongoose with TypeScript — Full Tutorial

- **Duración**: ~45 minutos
- **Idioma**: Inglés

## MongoDB Modelado de Datos

### MongoDB Data Modeling — MongoDB University
- **URL**: https://learn.mongodb.com/courses/m320-mongodb-data-modeling
- **Duración**: ~4-6 horas
- **Idioma**: Inglés
- **Descripción**: Curso oficial gratuito sobre patrones de modelado: embedding vs referencing, relaciones uno-a-uno, uno-a-muchos y muchos-a-muchos. Esencial para entender cuándo y cómo estructurar documentos.
