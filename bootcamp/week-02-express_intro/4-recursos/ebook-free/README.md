# Libros Gratuitos — Semana 02: Express Intro

## 📚 Recursos de lectura recomendados

### Node.js & Express

| Título | Enlace | Relevancia |
|--------|--------|------------|
| **Express.js Guide** (docs oficial) | https://expressjs.com/en/guide/routing.html | Routing y middleware — lectura obligatoria |
| **Node.js Design Patterns** (cap. HTTP) | https://www.nodejsdesignpatterns.com/ | Patrones de servidor HTTP en Node.js |
| **The Node.js Handbook** (Flavio Copes) | https://flaviocopes.com/books/ | Introducción a Node.js y Express, disponible gratis |

### HTTP y APIs REST

| Título | Enlace | Relevancia |
|--------|--------|------------|
| **HTTP: The Definitive Guide** (O'Reilly, preview) | https://www.oreilly.com/library/view/http-the-definitive/1565925092/ | Fundamentos del protocolo HTTP |
| **RESTful Web Services** (Leonard Richardson) | Disponible en bibliotecas digitales | Diseño de APIs REST con buenas prácticas |

---

## 💡 Consejo de lectura

Para esta semana se recomienda leer en este orden:

1. **Express.js Guide: Routing** — 15 min
2. **Express.js Guide: Writing Middleware** — 15 min
3. **Express.js Guide: Error Handling** — 10 min
4. **Node.js HTTP module** (docs.nodejs.org) — 10 min de comparativa

La lectura de libros completos es complementaria y opcional para profundizar conceptos.
