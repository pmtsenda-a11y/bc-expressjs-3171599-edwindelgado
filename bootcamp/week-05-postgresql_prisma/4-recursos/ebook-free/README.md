# Libros Gratuitos — Semana 05: PostgreSQL y Prisma

## 📖 Recursos Gratuitos

### PostgreSQL

**PostgreSQL Tutorial (online, gratuito)**
- URL: https://www.postgresqltutorial.com/
- Cobertura: fundamentos de SQL, DDL, DML, índices, joins, funciones, triggers
- Recomendado para: repasar SQL antes de avanzar con Prisma

**The Art of PostgreSQL** (capítulos de muestra gratuitos)
- URL: https://theartofpostgresql.com/
- Capítulos gratuitos: introducción al modelo relacional y SQL avanzado
- Autor: Dimitri Fontaine

### Prisma

**Prisma Docs — Getting Started** (documentación oficial, gratuita)
- URL: https://www.prisma.io/docs/getting-started
- Cobertura: setup completo, schema, migraciones, CRUD, relaciones
- Es la referencia más actualizada para Prisma 7

**Prisma Data Guide** (guías técnicas gratuitas)
- URL: https://www.prisma.io/dataguide
- Cobertura: conceptos de base de datos, SQL, tipos de relaciones, índices
- Muy útil para entender el modelo relacional desde la perspectiva de un desarrollador

### ORM y Bases de Datos

**Sequelize vs Prisma vs TypeORM** (comparativa gratuita)
- URL: https://prisma.io/docs/concepts/more/comparisons
- Cobertura: diferencias entre ORMs populares en Node.js
- Útil para entender por qué elegimos Prisma
