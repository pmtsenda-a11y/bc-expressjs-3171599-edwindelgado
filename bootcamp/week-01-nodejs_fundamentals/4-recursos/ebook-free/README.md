# 📚 Ebooks Gratuitos — Semana 01: Node.js Fundamentals

Recursos de lectura gratuitos para complementar el contenido de la semana.

---

## Node.js

### Node.js Notes for Professionals
- **Autor**: GoalKicker / Stack Overflow Documentation
- **Formato**: PDF gratuito
- **Descarga**: https://goalkicker.com/NodeJSBook/
- **Por qué leerlo**: Cubre eventos, streams, módulos, HTTP y más. Basado en respuestas reales de Stack Overflow. Excelente referencia para consulta rápida.
- **Capítulos relevantes esta semana**: 1 (Getting Started), 2 (Modules), 7 (Event Loop), 9 (Async/Await)

---

### Node.js Design Patterns (Muestra gratuita)
- **Autor**: Mario Casciaro & Luciano Mammino
- **Formato**: Primeros capítulos gratuitos en línea
- **Acceso**: https://www.nodejsdesignpatterns.com/
- **Por qué leerlo**: Cubre callbacks, Promises y patrones asíncronos con profundidad. Uno de los libros más referenciados en la comunidad Node.js.
- **Capítulos relevantes esta semana**: Cap. 1 (Node.js fundamentals), Cap. 3 (Callbacks & Events)

---

## TypeScript

### TypeScript Handbook (Oficial — gratuito online)
- **Autor**: Microsoft TypeScript Team
- **Formato**: Documentación web (también disponible como PDF unofficial)
- **Acceso**: https://www.typescriptlang.org/docs/handbook/intro.html
- **Por qué leerlo**: Referencia oficial y completa. Esencial para entender el sistema de tipos.
- **Secciones relevantes esta semana**: The Basics, Everyday Types, More on Functions, Object Types

---

### TypeScript Notes for Professionals
- **Autor**: GoalKicker
- **Formato**: PDF gratuito
- **Descarga**: https://goalkicker.com/TypeScriptBook2/
- **Por qué leerlo**: Referencia concisa con ejemplos prácticos. Buen complemento al handbook oficial.
- **Capítulos relevantes esta semana**: 1 (Getting Started with TypeScript), 3 (Classes), 5 (Interfaces)

## JavaScript Moderno (base)

### You Don't Know JS Yet (YDKJSY)
- **Autor**: Kyle Simpson
- **Formato**: Gratuito en GitHub
- **Acceso**: https://github.com/getify/You-Dont-Know-JS
- **Por qué leerlo**: Profundiza en el comportamiento real de JavaScript. Imprescindible para entender el Event Loop.
- **Libro relevante esta semana**: [Scope & Closures](https://github.com/getify/You-Dont-Know-JS/tree/2nd-ed/scope-closures), [Async & Performance](https://github.com/getify/You-Dont-Know-JS/tree/1st-ed/async%20%26%20performance)

---

### Eloquent JavaScript (4th Edition)
- **Autor**: Marijn Haverbeke
- **Formato**: Gratuito online / Descargable
- **Acceso**: https://eloquentjavascript.net/
- **Por qué leerlo**: Excelente introducción a JS moderno con capítulos dedicados a async programming.
- **Capítulos relevantes esta semana**: Cap. 11 (Asynchronous Programming)
