# 📚 eBooks Gratuitos — Semana 03

Recursos de lectura gratuita relacionados con arquitectura, patrones de diseño y APIs REST.

---

## Libros recomendados

### 1. Patterns of Enterprise Application Architecture — Martin Fowler

Referencia clásica sobre los patrones que usamos en el bootcamp: Repository, Service Layer, Data Mapper, Active Record.

- **Acceso gratuito a catálogo de patrones**: https://www.martinfowler.com/eaaCatalog/
- Los capítulos completos son de pago, pero el catálogo online es suficiente para el bootcamp.
- Patrones directamente relevantes esta semana:
  - [Repository](https://www.martinfowler.com/eaaCatalog/repository.html)
  - [Service Layer](https://www.martinfowler.com/eaaCatalog/serviceLayer.html)
  - [Data Transfer Object](https://www.martinfowler.com/eaaCatalog/dataTransferObject.html)

---

### 2. Node.js Design Patterns (3rd ed.) — Casciaro & Mammino

El mejor libro de patrones para Node.js moderno.

- **Capítulos relevantes**:
  - Capítulo 1: Node.js Platform (event loop, async I/O)
  - Capítulo 9: Behavioral Design Patterns (Strategy, Template Method)
- **Preview gratuito de capítulos**: https://www.nodejsdesignpatterns.com/
- El libro completo está disponible en O'Reilly Learning (prueba gratuita 10 días).

---

### 3. The REST API Design Handbook — George Reese

Guía práctica y concisa sobre diseño de APIs REST.

- **Descarga gratuita**: Busca en PDF Drive o Z-Library con el título exacto.
- Alternativamente: el artículo oficial de la disertación REST de Roy Fielding es gratuito:
  - https://www.ics.uci.edu/~fielding/pubs/dissertation/rest_arch_style.htm

---

### 4. TypeScript Deep Dive — Basarat Ali Syed

El mejor recurso gratuito para TypeScript avanzado, incluyendo generics, utility types y patrones.

- **Acceso gratuito y completo online**: https://basarat.gitbook.io/typescript/
- **Capítulos relevantes esta semana**:
  - Generics: https://basarat.gitbook.io/typescript/type-system/generics
  - Type Utility: Partial, Omit, Pick
